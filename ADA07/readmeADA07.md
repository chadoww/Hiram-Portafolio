# ADA07

## ¿En qué consistió?
La actividad consistió en implementar las tablas hash en dos aplicaciones distintas. La primera fue usando matrículas en direccionamiento abierto y la segunda como una tabla de símbolos de un compilador, se usó listas ligadas para manejar las colisiones.

## ¿Qué aprendiste?
Aprendí el funcionamiento de las tablas hash y como aplicar las estrategias de colisiones.

## Problemas o dificultades
Se me complicó las colisiones con las listas ligadas, porque implicaba controlar los punteros y no provocar fugas.
