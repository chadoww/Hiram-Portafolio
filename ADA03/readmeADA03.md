# ADA03

## ¿En qué consistió?
La actividad consistió en crear una lista doblemente ligada para almacenar películas del archivo Movie.csv, y programar funciones para cargar datos, buscar películas o actores, etc.

## ¿Qué aprendiste?
Principalmente aprendí a cómo manejar estructuras dinámicas con nodos doblemente enlazados, cómo recorrer y modificar una lista.

## Problemas o dificultades
Hubo mucho problema con el archivo csv, porque eran demasiadas las filas del archivo, que el programa no buscaba o no recorría todas con facilidad.
